About Project

This is an implementation of a sentiment analyzer, made for a Natural Language Processing class. I was tasked with recreating a Naive-Bayes classifier using only base Python packages. The model was trained on a set of 31kb of hotel reviews, and includes some custom optimizations with respect to calculating conditional probabilities of phrases. More explanation on the base functionality of the model coming soon.
